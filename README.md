# 危機就是轉鶏蛋糕 — 品牌官方網站

一個兼具現代視覺美學與流暢互動體驗的一頁式官網前端切版專案。
本專案以虛構的文青風雞蛋糕品牌為主題，整合多種前端開源套件實作，適合作為前端響應式網頁設計（RWD）的實作範例。

## 🛠️ 使用技術
* **核心基底：** HTML5 / CSS3 / SCSS (進行樣式客製化變更 `custom.css`)
* **UI 框架：** Bootstrap 5 (響應式網格系統與 Utility 通用類別)
* **字體與圖標：** Google Fonts / Bootstrap Icons
* **動態特效插件：** fullPage.js (實現全螢幕導航與視差錨點滾動)

## ✨ 專案亮點與技術實作

專案在架構上將龐大的頁面拆解為數個核心模組，並針對不同區塊進行了特化切版：

### 1. 模擬資源加載層
* 在主頁面渲染完成前，透過模擬進度條（Progress Bar Overlay）提供緩衝，優化使用者體驗（UX），避免因高解析度圖片未加載完成導致的畫面閃爍。

### 2. 全螢幕錨點導航
* 導入 `fullPage.js`，將網頁劃分為六大錨點區塊：`home`（首頁）、`brandstory`（品牌故事）、`products`（商品資訊）、`news`（最新消息）、`stores`（店鋪資訊）、`social`（社群連結）。
* 提供流暢的逐頁滾動體驗。

### 3. 輕量化商品輪播
* **亮點技術：** 捨棄了耗費效能的 JavaScript 輪播套件，**純粹使用 HTML `<input type="radio">` 結合 CSS 選擇器**控制 `carousel-track` 的位移，完成 9 款核心商品的無縫切換，大幅提升網頁的加載效能與流暢度。

### 4. 雜誌風格不對稱排版
* 在「Brand Story」區塊，利用 Bootstrap 網格與 CSS 絕對定位，挑戰非傳統的網格重疊排版，並利用 `backdrop-filter: blur()` 微光暈效果處理文字框，提升畫面層次感。

### 5. 動態列表與雙向跑馬燈
* 最新消息（News）區塊預留了滾動揭示（Reveal）動畫。
* 店鋪資訊（Stores）採用了雙向波浪滾動跑馬燈（Marquee Wave Track），營造動態活潑的視覺風格。

### 6. 響應式地圖整合與手風琴元件
* 在店鋪資訊（Stores）區塊，導入 Bootstrap Accordion（手風琴元件） 收納模擬 6 間分店明細，保持版面的整潔並優化垂直視覺動線。
